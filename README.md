# YNW

## Webpack 构建工具

- `Ynw Build this - Dev` : 使用`ynw-cli`开发模式构建
- `Ynw Build this - Hot` : 使用`ynw-cli`热更新模式开发
- `Ynw Build this - pro` : 使用`ynw-cli`生产模式构建

## 右键快捷菜单

> 以下右键菜单在设置中根据需要手动开启

- 导入当前选中模块(写入到当前打开的文档)
- 创建同名的文件
  - 如果选中的是 js 文件, 则创建同名的样式文件
  - 如果选中的是文件夹, 则创建同名的 js 文件
- 复制相对路径到剪贴板(相对当前打开的文档)
